# Day 1 - Linux (WSL Ubuntu)

## Commands I practiced
pwd
ls
ls -la
cd ~
mkdir -p ~/projects/cloud-notes/day01
cd ~/projects/cloud-notes/day01
touch notes.txt
echo "Day 1: Linux basics" > notes.txt
cat notes.txt
whoami
uname -a

## What I learned (short)
- pwd shows current directory
- ls lists files
- ls -la shows hidden files + permissions
- mkdir creates folders
- touch creates empty files
- echo writes text to a file
